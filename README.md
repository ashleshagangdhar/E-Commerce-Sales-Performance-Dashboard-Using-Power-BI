# E-Commerce-Sales-Performance-Dashboard-Using-Power-BI
Developed an interactive Power BI Sales Dashboard to analyze sales, profit, and 5K+ orders. Created KPI-driven visualizations for sales trends, regional performance, and top products. Used Power Query, DAX, and data modeling to transform data and enable interactive, data-driven decision-making.
## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Key Features
- Sales Trend Analysis
- Regional Performance Analysis
- Category-wise Performance
- Top Selling Products
- Interactive Filters and Slicers
- KPI Tracking Dashboard
